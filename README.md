# OmniCore Meter Suite - Updates

Auto-update distribution repository for OmniCore Meter Suite.

## Structure

Each release contains:
- `latest.json` — Update manifest for Tauri updater
- Windows installer (`.nsis.zip`) with signature
- Linux AppImage (`.AppImage.tar.gz`) with signature

## Usage

The application automatically checks this repository for updates on startup.
